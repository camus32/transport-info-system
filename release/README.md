# 铁路汽运信息录入系统

一个简洁高效的运输信息录入与管理工具，支持铁路和汽运两种运输方式的信息录入、列表展示和 TXT 导出功能。

## 功能特点

- 双模式录入：支持铁路运输和汽运运输两种模式
- 字段区分：
  - **铁路运输**：车号、挂车、姓名、证号、电话、箱号、箱型、到站、品名
  - **汽运运输**：车号、挂号、姓名、证号、电话、车型、到站、品名、发车时间、预计到达时间
- 记录管理：实时显示录入记录，支持单条删除
- 数据导出：一键导出所有记录为 TXT 文本文件
- 响应式设计：支持桌面和移动设备

## 技术栈

- React 18 + TypeScript
- Vite 构建工具
- Tailwind CSS 样式框架
- shadcn/ui 组件库

## 使用说明

1. 选择运输模式（铁路/汽运）
2. 填写相关信息
3. 点击"添加记录"按钮
4. 在列表中查看已添加的记录
5. 点击"导出TXT"按钮导出所有记录
